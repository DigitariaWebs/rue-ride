## Quick Reference: Icon Style Guide

**Style**: Minimal, modern, consistent stroke width (2px)
**Format**: SVG with viewBox
**Colors**: Use CSS variables for theming
**Sizes**: 24x24px default, provide 48x48px for large variants

## Color Extraction from Inspiration

To extract colors from `inspiration.webp`:

1. **Using Image Editor** (GIMP, Photoshop, Figma):
   - Open the image
   - Use eyedropper tool to sample colors
   - Note hex/RGB values for:
     - Primary brand color
     - Secondary/accent colors
     - Background colors
     - Text colors

2. **Using Online Tools**:
   - Upload to: https://imagecolorpicker.com/
   - Or: https://coolors.co/image-picker

3. **Update PROJECT_PLAN.md** with extracted colors

## Asset Sources

**Icons**: 
- Can use Lucide React (already installed) for standard icons

**Illustrations**:
- Create in Figma/Illustrator
- Or use illustration libraries (unDraw, Storyset)
- Or commission custom illustrations

**Images**:
- Use stock photos (Unsplash, Pexels) with Paris theme
- Ensure proper licensing
- Optimize with tools like Squoosh or ImageOptim
